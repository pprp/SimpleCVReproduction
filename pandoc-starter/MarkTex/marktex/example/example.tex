\section{特性\textsubscript{下标在这里}}


\end{marktext}
\begin{itemize}
\item
支持目前主流的所有markdown语法（目前，脚注和xml标签暂时不支持）
\item
额外添加了下划线语法（\underline{下划线}）
\item
表格自动调整列宽
\item
复选框支持三种
\item
无论是本地图片还是网络图片，都能够支持
\end{itemize}

\subsection{表格}


支持一般的文本格式，暂时不支持表格内图片。另外，表格取消了浮动（float），因此不支持对表格的描述（caption），不过在Markdown中也没有对表格的描述，因此也不算功能不完善。
